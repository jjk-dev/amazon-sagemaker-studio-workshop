Export data

In this lab, you will export your transformed data. Data Wrnagler easily integrates that data flow into your data processing pipeline.

Export data flow

  1. Choose the Export tab and do right click on Steps icon. Select the last step in your data flow.

  2. Export step button will appear. Click the button and choose Data Wrangler Job.

  3. Then a Jupyter Notebook is automatically created to run a SageMaker processing job to execute your data flow. Choose Python 3 (Data Science) for the Kernel to start this notebook.

  4. When the kernel starts, change following variables under Output: S3 settings part.

  • bucket = “Change the s3 bucket name which you made." (e.g. “sagemaker-accountId”)

  1. Run cells until Job Status & S3 Output Location part. After the processing job fininshed, check the result. This will take about 10 mins.

  2. To see the transformed result, run (Optional) Load Processed Data into Pandas part.

  3. Go to S3. And confirm that transformed data made as csv file in your S3 bucket.
