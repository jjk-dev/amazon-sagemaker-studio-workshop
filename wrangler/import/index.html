Import data

In this lab, you will import dataset to Data Wrangler.

  1. Rename the untitled.flow to banking_fraud .flow using right click.

  2. Select Amazon S3 to import banking fraud dataset.

  3. Click your S3 bucket name that we made in Prerequisite step.

  4. And select banking_fraud.csv file then click Import dataset button on the Details panel. You can see some of your data in Preview panel.

  5. The tab moved to Prepare tab. In this section, your dataset appears in the data flow.
