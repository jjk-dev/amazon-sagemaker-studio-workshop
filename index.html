AMAZON SAGEMAKER STUDIO HANDS ON LAB

What is Amazon SageMaker Studio?

Amazon SageMaker Studio is the first fully integrated development environment (IDE) for machine learning that provides a single, web-based visual interface to perform all the steps for ML development.

Amazon SageMaker provides every developer and data scientist with the ability to prepare build, train, and deploy machine learning (ML) models quickly. SageMaker removes the heavy lifting from each step of the machine learning process to make it easier to develop high quality models. SageMaker provides all of the components used for machine learning in a single toolset so models get to production faster with much less effort and at lower cost.

Overview of the labs

On this tutorial, you learn how to use Amazon SageMaker Studio, SageMaker Data Wrangler, and SageMaker Studio Notebook to prepare data, build and train a machine learning (ML) model.

This takes about 1 hour and contains:

  1. Prerequisite
  2. SageMaker Data Wrangler
  3. SageMaker Studio Notebook
  4. Clean up

Every step is connected to each other. So please follow the tutorial step by step.
